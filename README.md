# Traffic Data Analysis Program (Python)

## Overview
A Python application developed to analyse real-world traffic datasets (CSV format) for a local council scenario.  
The program validates input data, processes multiple datasets, and generates statistical summaries and visualisations.

## Features
- CSV file input and validation
- Automated statistical analysis
- Histogram visualisation of traffic volumes
- Generation of text-based summary reports
- Error handling

## Technologies
- Python 
- graphics.py

## How to Run
1. Clone the repository  
2. Run `python main.py`  
3. Select a dataset CSV file when prompted  

## Output
- Console summary statistics  
- Histogram visualisation  
- Text file containing summary results
